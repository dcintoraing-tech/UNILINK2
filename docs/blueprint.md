# **App Name**: UniLink Access

## Core Features:

- User Registration: Allow new users to create an account with an email and password, following La Uni's brand identity for form inputs and buttons.
- Secure Login: Provide a login form for existing users to securely access the platform using their credentials, maintaining the visual style of La Uni.
- Password Recovery Flow: Implement a 'Forgot Password' functionality to help users reset their password, aligning the UI with La Uni's visual guidelines.
- Session Management: Enable user sessions for persistent login and provide a clear logout mechanism, styled to integrate seamlessly with the branding.
- Simple User Dashboard: After successful login, present a minimalist dashboard or welcome page displaying basic user information, adhering to La Uni's aesthetic.

## Style Guidelines:

- Primary color: A vibrant teal (#00A99D) derived directly from La Uni's prominent branding, representing connectivity and progress, used for key interactive elements like buttons and active states.
- Background color: A very light, subtle teal (#E7F7F6), providing a clean and refreshing canvas that complements the primary brand color while ensuring legibility in a light theme.
- Accent color: A rich green (#1EB859) used for secondary calls to action, important notifications, or subtle highlights, providing a harmonious yet distinct contrast with the primary teal.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics, aligning with La Uni's contemporary and professional image for all textual content.
- Use minimalist, rounded line-art icons that maintain a clean and professional appearance, consistent with modern UI trends seen on La Uni's website.
- Design a clean, centered layout for the login and registration forms, prioritizing user experience with ample whitespace and intuitive element placement, responsive across device sizes.
- Implement subtle hover effects on interactive elements and smooth transitions for form input focus states to enhance user engagement without being distracting.
